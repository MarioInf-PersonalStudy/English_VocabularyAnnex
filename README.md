# English-Vocabulary-Annex

This Repository contents a compilation of a English vocabulary shorted by topics. You can use it learn a general topics or to prepare a official certification.

If you only want to get the whole Annex, you have to download the main [PDF](./English_VocabularyAnnex.pdf).

## Prerequisites

This project is prepared to run using [LaTeX](https://www.latex-project.org/) in a [Visual Studio Code](https://www.tug.org/texlive/windows.html#install), and is free to be modified by a third person. If you want to do changes, you have to follow the next steps before run the application:

1. Install a **_LaTeX Compiler_**, depending your Operative System:

    - Using **_Debian/Ubuntu_**:

        ```bash
        sudo apt update
        sudo apt install texlive-full
        ```

    - Using **_Fedora_**:

        ```bash
        sudo dnf install texlive-scheme-full
        ```

    - Using **_Arch Linux_**:

        ```bash
        sudo pacman -S texlive-most
        ```

    - Using **_Windows_**: You have to download and run the [LaTeX Instalador fow Windows](https://www.tug.org/texlive/windows.html#install).

2. Install **_VS Code_** following the instructions in the [Official Page](https://code.visualstudio.com/).

3. We recommend using some extension to facility the use of **_LaTeX_** in **_VS Code_**. We recommend use the following:
    - [LaTeX Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop).
    - [LaTeX Utilities](https://marketplace.visualstudio.com/items?itemName=tecosaur.latex-utilities).
